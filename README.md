## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run dev

# production mode
$ pnpm run prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run e2e

# test coverage
$ pnpm run cov
```
